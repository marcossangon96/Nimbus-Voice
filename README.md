# Nimbus Voice

Nimbus Voice is an AI-powered voice interface that lets you **turn written text into natural, expressive speech instantly** using state-of-the-art generative AI models.

It combines large language models with high-quality text-to-speech to make interacting with AI feel more human and fluid.

## How It Works

1. **Type a message** in the input box.
2. **Select a voice** from the available options.
3. **Click Send**.
4. The message is processed by **Vertex AI** to generate a response.
5. The response is converted into speech using **ElevenLabs Text-to-Speech**.
6. Listen to the audio and read the generated text directly in your browser.

## Features

- AI-generated responses in real time.
- Natural and expressive text-to-speech voices.
- Simple and intuitive user interface.
- Fully browser-based, no installation required.
- Suitable for demos, storytelling, accessibility use cases, or voice-driven experiences.

## Interface Overview

- **Text Input**: Write the message you want the AI to respond to.
- **Voice Selector**: Choose the voice used for speech output.
- **Send Button**: Submits the message and triggers audio generation.
- **Audio Player**: Plays the generated voice output.
- **Response Panel**: Displays the text generated by the AI.

## Usage Tips

- Shorter messages generate faster responses.
- Try different voices to explore tone and expressiveness.
- Long responses can be scrolled in the text panel.
